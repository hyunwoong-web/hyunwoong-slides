Framework

시너지의 3가지 유형 — 사례가 각각 다른 축을 증명

"브랜드 소재를 퍼포먼스에 썼다"는 한 문장이 아니라, 실제로는 서로 다른 세 가지 협업 메커니즘이 존재. 3사가 각 유형의 대표 사례.
A

소재 시퀀싱 · 리타겟팅

브랜드 필름을 쪼개 예고편 → 본편 → 액션 순서로 노출. 브랜드 자산이 리타겟팅 모수(시청자 풀)를 만들고, 퍼포먼스가 그 풀을 수확.

대표: 토스 — 예고편 시청자 리타겟팅으로 50분 본편 시청 유도

·
B

반응 데이터 → 소재 변주

브랜드 소재의 시청·댓글 반응을 퍼포먼스식 소재 실험 데이터로 취급. 반응이 좋은 방향으로 파생 소재를 대량 재생산.

대표: 여기어때 — 1시간 버전·노동요·노래방 버전, 댓글 기반 모델 캐스팅

·
C

브랜드 캠페인의 그로스식 측정

인하우스 데이터팀이 브랜드 캠페인에 증분(incrementality) 측정 프레임을 적용 — 집행 중 실시간 최적화로 미디어 효율 개선.

대표: 오늘의집 — TVC 슬롯별 AU/Install 증분 대시보드

A·B는 크리에이티브 자산의 활용, C는 측정·운영 체계의 이식 — 셋 다 "브랜드팀 따로, 퍼포먼스팀 따로"를 깨는 구조.

재시청 36%의 의미 — 도달이 아니라 기억 점유

일반 브랜드 캠페인의 KPI는 도달·조회단가. 재시청률은 자발적 소비 = 기억 점유(mental availability)의 지표로, "여행 순간이 왔을 때 검색·설치로 이어지는 풀"을 만드는 상위 퍼널 본연의 역할을 정량으로 보여줌.

Context

왜 이 사례가 중요한가

토스·여기어때가 브랜드 소재의 활용을 보여준다면, 오늘의집은 브랜드 캠페인을 운영하는 체계를 보여줌. 대행사에 맡기고 결과 보고서를 받는 구조가 아니라, 인하우스 데이터팀이 측정 프레임을 직접 설계하고 집행 중 최적화까지 수행.

High-involvement

리빙·인테리어 = 고관여 + 저빈도

가구·시공은 탐색 기간이 길고 객단가가 높은 고관여 카테고리 — 부동산과 구조 유사. 순간 수요가 왔을 때 떠오르는 브랜드가 되기 위한 상위 퍼널 투자를, 퍼포먼스 조직의 언어(증분·대시보드)로 정당화한 사례.

이벤트 드리븐 · 고관여 서비스 관점
단기임대도 여행·리빙과 같은 저빈도·고관여·이벤트 드리븐 구조 — 3사의 메커니즘이 그대로 이식 가능.
A · 토스식

히어로 소재를 시퀀싱 단위로 기획

브랜드 필름을 만들 때부터 예고편·컷다운·본편을 세트로 설계하고, 시청자 풀을 리타겟팅 모수로 축적 → VVC·DemandGen 오디언스로 수확. 상위 퍼널 확장(SL 상위퍼널 전략)과 직결.

B · 여기어때식

반응 데이터를 소재 파이프라인에 연결

브랜드 소재의 댓글·재시청·유지율을 소재 실험 데이터로 취급해 반응 좋은 방향으로 파생 소재를 재생산 — 현재 운영 중인 소재 대량 투입 체계(설치 소재 12→111개 경험)와 결합.

C · 오늘의집식

집행 전에 증분 측정 프레임부터

브랜드 캠페인 승인 조건으로 베이스라인·증분 지표·실시간 대시보드를 먼저 정의. MMP(Airbridge)+EMM 파이프라인이 이미 있어 이식 비용 낮음 — Brand Lift·Conversion Lift 병행.

순서 제안: C(측정 프레임) → A(소재 시퀀싱 설계) → B(반응 루프). 측정 없는 브랜드 캠페인이 가장 비싼 실패 — 오늘의집이 첫 TVC에서 증명한 교훈.

조사 한계 · Q&A 대비

정직한 각주

"브랜드 소재 재활용 → ROAS +X%" 단일 공식 사례는 없다

브랜드 크리에이티브를 전환 캠페인 소재로 그대로 재활용해 ROAS 개선을 공식 수치로 공개한 한국 사례는 공개 자료에서 확인되지 않음. 본 덱은 대신 시너지의 3가지 메커니즘(시퀀싱·소재 변주·증분 측정)을 각각 공식/1차 소스로 증명하는 구성.

부동산 카테고리 공식 사례 부재

직방(전지현 캠페인 2주 3,500만 뷰 · 2025)은 카테고리 적합도가 높으나 플랫폼 공식 성공사례·퍼포먼스 연계 수치가 미공개라 본편에서 제외. 필요 시 "카테고리 권위 구축" 관점의 보조 사례로 추가 가능.

상관 ≠ 인과

여기어때 신규 설치 1위, 오늘의집 다운로드 성장은 브랜드 캠페인 단독 효과로 분리 검증된 수치가 아님(프로모션·제품·시장 성장 동시 작용). 인과 수준의 근거는 토스 브랜드 리프트(+5%/−65%)와 오늘의집 증분 프레임 방법론.
