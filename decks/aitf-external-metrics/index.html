Q&A

"사람이 못 보던 것을 매일 보고,
울리면 이유까지 온다"

Q1알림이 너무 많아지면?

소음 제어 6원칙 + 임계 튜닝 — 실적: R4 페이스 7→2건, GA4 12→1건, 콘텐츠 17→0건. 임계값은 상수라 조정이 숫자 1개

Q2수집이 끊기면?

3중 방어 — 실패 자기 보고(Chat) + 회사 단위 수집 중단 감지 + 워치독 재기동

Q3다른 도메인 확장 가능?

수집기 플러그인 구조 — 새 지표 = 수집기 1개 + 감지 규칙 추가. 파이프라인·라우팅·소음 제어는 공통 인프라

맺음말 — LLM 소견 단계 공동 설계 등 AI TF와의 협업을 제안합니다
